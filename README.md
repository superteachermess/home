# 🚀 Teacher Mess | Resgate sua Fluência

Este é o repositório da Landing Page oficial do **Teacher Mess**, uma plataforma de mentoria focada em resgatar a fluência e a confiança de alunos que já possuem uma base de inglês, mas sentem-se "enferrujados".

## 🎯 Sobre o Projeto

A página foi desenvolvida com foco em **alta conversão (CRO)** e **experiência do usuário (UX)**. Utiliza uma estética moderna "dark mode" com elementos em vidro (glassmorphism) e animações suaves para transmitir profissionalismo e dinamismo.

### O Método RESCUE

O site apresenta a mentoria de Messias Rabello, focada em:

* **Recuperação de Confiança:** Para quem já estudou e parou.
* **Prática Ativa:** Grupo VIP e interação diária.
* **Aprendizado Dinâmico:** Uso de humor e recursos visuais para fixação.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **HTML5 Semântico:** Para melhor indexação em motores de busca (SEO).
* **Tailwind CSS:** Framework utilitário para estilização rápida e responsiva.
* **JavaScript (Vanilla):** Lógica customizada para animações, FAQ, menu mobile e scroll suave.
* **Font Awesome:** Biblioteca de ícones vetoriais.
* **Google Fonts:** Tipografia *Poppins* para leitura clara e moderna.

---

## ✨ Funcionalidades

* **Design Totalmente Responsivo:** Adaptado para smartphones, tablets e desktops.
* **Menu HUD Fixo:** Navegação intuitiva com efeito de transparência (backdrop-blur).
* **Animações de Scroll (Reveal):** Elementos surgem conforme o usuário navega pela página.
* **FAQ Interativo:** Seção de dúvidas frequentes com sistema de acordeão.
* **Scroll Spy:** O menu destaca automaticamente a seção em que o usuário se encontra.
* **Integração com Hotmart:** Botões de checkout configurados para conversão direta.
